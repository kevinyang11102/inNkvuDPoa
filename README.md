# 图书馆管理系统简介 python151

## 项目概述

图书馆管理系统是基于Python语言，采用Flask框架和MySQL数据库构建的一套软件应用。系统分为管理端与用户端，提供用户登录、注册，图书信息增删改查等功能，并包含可视化与采购申请模块，便于拓展和二次开发。

## 技术栈

- **开发语言**：Python
- **后端框架**：Flask
- **数据库系统**：MySQL
- **开发工具**：PyCharm
- **其他工具**：Navicat（数据库连接工具）

## 功能模块

- 用户登录与注册
- 图书信息增加、修改、删除、查询
- 可视化功能
- 用户申请采购模块

## 系统角色

- 管理员：负责管理图书信息，处理用户采购申请等操作。
- 用户：可以进行图书查询、申请采购等操作。

## 运行环境

- **操作系统**：Windows、Linux

## 部署步骤

1. 使用PyCharm导入项目。
2. 安装所需依赖包。
3. 配置MySQL数据库。
4. 正常启动应用。
